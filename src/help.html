How to Play MINESWEEPER :D

Overview

Minesweper is a luck and stradegy game in which the play tries to clear the board.
The board is a grid which starts out covered, with some of the celss containing mines
underneath. The player clicks on a cell to reveal it. If the player reveals a mine, he is
dead and the whole board will be shown. If not, the player can continue to rveal cells untill
all the cells not contained mines are revealed

Types of Cells

An unrevealed cell




A numbered cell. Theses cells tell u how many bombs are around them. Each cell can have up to 8 bombs around them.




A flagged cell. You flag a cell to remind yourself that it contains a bomb.




A undetermined cell. You can add a ? to a cell if you dont know if its a bomb or not.




A bombed cell. If you click on a bomb, the bomb will show up like this when the whole board is shown




A bomb cell. A cell that contains bomb will be displayed like this when the whole board is shown




A clear cell. A cell that is incorrectly flagged will be displayed like this when the whole board is shown
